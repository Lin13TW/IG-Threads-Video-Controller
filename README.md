# Threads Video Controller

Chrome 擴充套件，為 Threads 和 Instagram 網頁版加入自訂影片控制器。

> 本專案由 AI 協助生成與維護。

## 功能
- 播放 / 暫停
- 進度條拖曳
- 調整播放速度（0.5x ~ 3x）
- 靜音 / 音量控制
- 畫面旋轉（每次 90°）
- 支援 Threads 和 Instagram

## 安裝方式

1. 到右側 [Releases](../../releases) 下載最新的 `.zip` 檔
2. 解壓縮，得到一個資料夾
3. 打開 Chrome，網址列輸入 `chrome://extensions`
4. 右上角打開「**開發人員模式**」
5. 點「**載入未封裝項目**」，選擇剛剛解壓縮的資料夾
6. 完成！去 Threads 或 IG 開啟影片試試看
